# LeetCode刷题记录

牛顿迭代法求解以下方程的根：
$$
f(x)=x^2-C
$$
迭代式为：
$$
x_{n+1} = x_{n}-\frac{f(x_n)}{f^{'}(x_n)} = x_n-\frac{x_n^2-C}{2x_n}=\frac{x_n+\frac{C}{x_n}}{2}
$$

```c++
class Solution {
public:
    int mySqrt(int x) {
        if (x == 0) return 0;
        if (x == 1) return 1;
        long long a = x;
        while (a > x / a) {
            a = (a + x / a) / 2;
        }
        return a;
    }
};
```
